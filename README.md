# 4SPI_SSD1322_LIB
SSD1322 should be connected to Arduino Mega 2560 like:  
SSD1322 <---> Arduino Mega 2560  
GND <---> GND  
VCC <---> 3.3V  
RES <---> D10  
CS <---> D11  
D/C <---> D12  
E/RD <---> GND  
R/W <---> GND  
D0 <---> D52  
D1 <---> D51  
D2 <---> Not connected  
D3 <---> GND  
D4 <---> GND  
D5 <---> GND  
D6 <---> GND  
D7 <---> GND
